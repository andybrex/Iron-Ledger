# Iron Ledger

A personal workout tracker — build blocks of exercises, run them as a guided
timeline with rest timers, log your reps, and track your progressive-overload
trend over time.

Single self-contained file (`index.html`). No build step: React, Tailwind, and
Babel load from CDNs, and data is saved in the browser via `localStorage`.

## Deploy
Hosted as a static site on Vercel — every push to `main` redeploys automatically.
